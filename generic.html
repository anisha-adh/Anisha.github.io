  • Mushoor Home Harvest System

    • This is a climate-controlled device which can maintain temperature, humidity, and air circulation inside a closed container. The module comprises two major parts: The controller box and the container itself. The controller takes input of temperature, humidity and soil moisture from multiple sensors inside the container and actuates heater and humidifier based on the feedback.Inside the box there are multiple racks on four levels for putting trays where the mushrooms can be grown. As per calculated this box consumes on average of 130W of energy. We have used 40mm thick Sandwitch(PUF panel) to construct the outer cover of box. Firstly it includes the basic IOT features like remote monitoring and controlling of its parameters like temperature, humidity, etc. This is not only limited to just monitoring using a remote mobile device but in order to get optimal production, users are constantly provided with firmware updates which can be installed with just one click. These updates are equipped with latest research data from the academic research institutions who are constantly engaged into the study of better productivity of mushrooms. This way, we link top mushroom scientists with the local farmers directly. We take the data directly from our research module installed and apply machine learning to learn what is the best scenario for which type of mushroom and send the OTA updates directly to the end users.

  • Magic Mirror

    • The time, weather, calendar, news, and social media updates are all displayed on a smart mirror, commonly referred to as a magic mirror. The magic is created by placing a transparent mirror over a screen such as a monitor. Voice recognition by AVS SDK is paired with a Raspberry Pi or Windows PC to power the system. Everything was controlled by Raspberry Pi.
